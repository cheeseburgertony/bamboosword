# Welcome to BambooSword

BambooSword 是一个基于 React 和 TypeScript 打造的现代组件库。它旨在提供高质量的 UI 组件，帮助开发者快速构建漂亮的 Web 应用。

### 安装试试
~~~bash
npm install bamboosword --save
~~~

### 使用
~~~javascript
// 加载样式
import 'bamboosword/dist/index.css'
// 引入组件
import { Button } from 'bamboosword'
~~~
